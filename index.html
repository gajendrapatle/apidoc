Login API

Endpoint: POST /api/login

Request

curl -X POST https://api.example.com/api/login -H "Content-Type: application/json" -d '{"data":{"username":"your_username","password":"your_password"}}'

Response

{
  "status": "success",
  "data": {
    "access_token": "eyJhbGciOi..."
  },
  "messages": []
}

Description

FieldTypeDescription
usernameStringUsername provided by StARMF system
passwordStringPassword provided by StARMF system
