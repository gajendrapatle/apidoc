🔐 Login API

Endpoint: POST /api/login

curl -X POST https://api.example.com/api/login \
-H "Content-Type: application/json" \
-d '{"data":{"username":"your_username","password":"your_password"}}'
{
  "status": "success",
  "data": {
    "access_token": "eyJhbGciOi..."
  },
  "messages": []
}
FieldTypeRequiredDescription
usernameStringYesAPI username
passwordStringYesAPI password

👤 Add UCC API

Endpoint: POST /v2/add_ucc

curl -X POST https://api.example.com/v2/add_ucc \
-H "Content-Type: application/json" \
-H "Authorization: Bearer <token>" \
-d '{ "data": { "member": {}, "clientcode": "001", "holders": [] } }'
FieldTypeRequiredDescription
memberObjectYesMember details
clientcodeStringYesInvestor UCC
holdersArrayYesList of holders

📦 Order New API

Endpoint: POST /v2/order_new

curl -X POST https://api.example.com/v2/order_new \
-H "Authorization: Bearer <token>" \
-d '{ "data": { "order_type": "BUY", "clientcode": "001" } }'
